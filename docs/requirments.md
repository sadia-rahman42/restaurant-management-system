# Restaurant Management System

## Project Overview

Restaurant Management System is a web-based application designed to manage the daily operations of a restaurant. It helps restaurant staff handle orders, tables, menus, billing, customers, and reports efficiently.

---

## Project Goal

Build a modern, secure, and user-friendly restaurant management system using HTML, CSS, JavaScript, Python, and MySQL.

---

## User Roles

1. Admin
2. Manager
3. Cashier
4. Waiter
5. Chef
